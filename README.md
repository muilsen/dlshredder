# dlshredder (v1.1.0)
<sup>dlshredder - download shredder</sup>

a tool made to make your Downloads folder less messy.<br>
## Current features:
- Shredding (deleting) files inside the specified Downloads folder
- Quarantining (moving) files to a different (custom user-specified) folder
- Displaying all files inside the specified Downloads folder
- Displaying file sizes in either MB or MiB

## Instructions
### Setting up the Downloads folder
1. Click on the Folder menu in the top left corner
2. Select "Set Downloads Folder"
3. Select your Downloads folder
(selecting the Downloads folder is needed to actually use the program)<br>

### Setting up the Quarantine folder (OPTIONAL)
1. Click on the Folder menu in the top left corner
2. Select "Set Quarantine Folder"
3. Select the folder you want to use as your Quarantine Folder<br><br>
(setting up the Quarantine Folder is not necessary but doing so gives access to the "Quarantine File" feature)
